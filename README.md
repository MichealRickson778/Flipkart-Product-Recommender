# Flipkart-Product-Recommender
Flipkart Product Recommender using Prometheus, Grafana, Minikube, AstraDB, Langchain, Groq, Huggingface 

# Tech Stack:

1. Groq - LLM
2. HuggingFace - Embedding Model
3. GCP VM - to deploy and run your app on the cloud with in the virtual machine
4. Langchain - Genratibe AI framework to interact with LLM
5. Minikube - Making a kuberntes cluster inside a VM
6. Docker - for containerization of the app during deployement 
7. Flask - For making backdn of your application
8. HTML/CSS - to make UI/Frontend of the app
9. Kubectl - CLI tool for kubernets to interact with machine
